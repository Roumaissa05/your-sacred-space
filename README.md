# 🕊️ Your Sacred Space | مساحتكِ االامنة

A calming bilingual self-care web experience designed to provide emotional comfort, breathing exercises, affirmations, and peaceful interactions.

## ✨ Features

- 🌸 Beautiful soft aesthetic UI
- 🌙 Arabic & English language support
- 💖 Random emotional quotes
- 🧘 Breathing meditation exercise
- 🎵 Relaxing music player
- 🌿 Interactive affirmations
- 📱 Fully responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🎯 Purpose

This project was created as a peaceful digital space for emotional relaxation and mindfulness.
